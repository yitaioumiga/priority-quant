# 项目优先级量化评估

## 任务目标
- 本 Skill 用于: 通过双轨制评估模型（极速4象限/深度6维度）量化项目优先级，结合战略对齐检验和工作流闭环，提供智能化的优先级决策支持
- 能力包含: AI预审推荐、战略对齐检验、双轨制评估、飞书多维表格集成、WBS拆解触发
- 触发条件: 用户需要进行项目优先级排序、资源分配决策、项目选型或日常任务管理

#### 自主智能体功能

**OpenClaw** 支持以下自主任务：

- **周度项目优先级复盘**：每周一自动收集新增项目、评估优先级、生成报告
- **月度项目优先级报告**：每月初汇总项目、分析趋势、提供资源分配建议
- **新项目入库评估**：新项目提交时自动执行6维度评分、检查战略对齐度

**Hermes** 支持更深层次的自主任务：

- **周度项目优先级复盘**：支持并行评估、可视化报告、智能提醒
- **月度项目优先级报告**：深度分析、识别瓶颈、提供优化建议
- **新项目入库评估**：AI预审推荐、批量处理、自动通知
- **战略对齐度检查**：定期检查所有项目与核心目标的对齐度

## 前置准备
- Python 3.8+ 环境
- 准备待评估项目的基本信息（背景、目标、时间要求等）
- 可选: 飞书多维表格访问权限（用于导出结果）

## 操作步骤
- 标准流程:

  **步骤1: 模式识别与战略对齐**
  - 识别项目复杂度，自动选择评估模式:
    * Fast-track（极速模式）: 日常小项目、单人任务、<1周工作量
    * Deep-track（深度模式）: 复杂项目、多方协作、≥1周工作量
  - 战略对齐检验:
    * 询问或确认用户当前核心目标
    * 验证项目与战略的对齐性
    * 若项目严重偏离核心目标，给出警告并建议重新评估

  **步骤2: 分支处理**

  **分支A: Fast-track（极速模式）**
  - 评估方式: 重要/紧急4象限快速分发
  - 输出: 优先级分类 + 快速行动建议
  - 跳至步骤3

  **分支B: Deep-track（深度模式）**
  - AI预审推荐:
    * 智能体基于项目描述预估6个维度分数
    * 提供每个维度的预估分数和打分理由
  - 用户复核微调
  - 脚本精算:
    ```bash
    python scripts/evaluate.py --projects '[{"name":"项目A","urgency":5,"importance":5,"scale":3,"feasibility":4,"roi":4,"dependencies":2}]'
    ```
  - 结果分析与行动建议
  - 跳至步骤3

  **步骤3: 工作流闭环**
  - 飞书集成（可选）
  - WBS拆解触发（针对Top1项目）
  - 数据持久化

| 重要程度 | 数字 | 1-5分 | 5 |
| 规模大小 | 数字 | 1-5分 | 2 |
| 实现可能性 | 数字 | 1-5分 | 4 |
| ROI预期 | 数字 | 1-5分 | 5 |
| 依赖阻塞 | 数字 | 1-5分 | 3 |
| 加权总分 | 数字 | 脚本计算结果 | 4.1 |
| 排名 | 数字 | 排名结果 | 1 |
| 状态 | 单选 | 待评估/评估中/进行中/已完成/暂停 | 评估中 |
